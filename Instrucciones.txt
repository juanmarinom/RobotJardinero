Instrucciones para la instalación:
1. Instalar Python 3.6
2. Tener Python en el Path de Windows como python
3. Ejecutar desde la consola python installer.py desde la carpeta del programa.
	# En caso de no instalarse la libreria tensorflow instalar manualmente.
4. Una vez instalado el entorno virtual ya se puede ejecutar todos los scripts de python
5. Instalar VREP y que los archivos .ttt se abran directamente con la aplicación.
6. Copiar remoteApi.dll en la carpeta del programa en caso de que no funcione desde la ruta de VREP

Instrucciones de uso:
1. Ejecutar el script de program.py desde la ruta del programa.
